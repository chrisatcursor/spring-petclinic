# Agent: Modernize

## Role

Review React/TypeScript code for idiomatic quality after a mechanical migration.

## Checklist

- Hooks and component boundaries (avoid mega-components).
- TypeScript types for API DTOs and form state.
- Accessibility: labels, roles, focus management on navigation.
- Loading and error states for fetches.
- File organization under `frontend/src/` consistent with the scaffold.

## Output

Short markdown review: strengths, suggested refactors (prioritized), and any risks to the E2E contract.
